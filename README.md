# 개장 전 브리핑 (Premarket Briefing)

매일 평일 아침 07:00 KST에 자동으로 실행되는 한국 증시 브리핑 파이프라인.

**원칙: 파이프라인은 하나, 출력은 세 개.**

| 출력 | 내용 | 전달 |
| --- | --- | --- |
| 풀 리포트 | 예측·전략 포함 (비공개) | 이메일 전용, 레포에 커밋되지 않음 |
| 공개 브리핑 | 예측 제거된 팩트 브리핑 | GitHub Pages 자동 게시 |
| 네이버용 페이지 | 네이버 형식 복붙 원본 | 비공개 URL (noindex), 수동 복붙 발행 |

## 구조

```
docs/prompts/   # 사양: v3 리서치 프롬프트, 블로그 변환 프롬프트
pipeline/       # Python 파이프라인 (리서치 → 검증 → 변환 → 이미지 → 게시/메일)
site/           # Astro 블로그 (GitHub Pages)
data/           # KRX 휴장일 등 정적 데이터
reports/        # 풀 리포트 로컬 아카이브 (gitignore, 커밋 금지)
.github/        # Actions 워크플로우
```

## 파이프라인 흐름

휴장일 체크 → Step 1 리서치(Claude + 웹서치) → 검증 게이트(티어1 수치 8종)
→ Step 2 블로그 변환(+괴리율 코드 재검증) → Step 3 이미지 → Step 4 게시 + 이메일

어느 단계든 실패하면 **게시하지 않고** 오류 이메일만 발송한다.
("잘못된 글이 나가는 것 < 하루 거르는 것")
